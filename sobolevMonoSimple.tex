\subsection{Случай веса, не зависящего от $u$}
\label{sobolevMonoSimple}

В данном случае условия (\ref{pieceMono}) и (\ref{flatMin}) можно
переформулировать так: функция $a(x)$ кусочно-монотонная и в каждом ненулевом
минимуме имеет площадку постоянства. Условие же (\ref{wideZero}) будет
выполняться автоматически.
Процесс избавления от условия (\ref{pieceMono}) будет изложен позднее.
Обсудим здесь, как избавиться от условия (\ref{flatMin}).

Положим, функция $a(x)$ кусочно-монотонна. Нашей задачей будет приблизить ее
снизу функциями, удовлетворяющими (\ref{goodWeight}).
Введем обозначение для множества нулей функции $a$: $Z = a^{-1}(0)$ и для
множества локальных строгих ненулевых минимумов функции $a$
$M = \{ x \in ( -1, 1 ) \setminus Z: \exists \eps( x ) \forall y :
0 < \abs{ x - y } < \eps \ a( y ) > a( x ) \}$.
Пусть $a_g$ ($g \ge 0$) это функция, полученная из $a$ заменой на
константу в окрестностях этих точек так, что сам минимум поднимается на $g$.
А именно: пусть $x \in M$ и $I_x = ( l_x, r_x ) \ni x$ такой интервал, что
$a\mathop{|}_{I_x} < a( x ) + g$ и
$a( l_x ) = a( r_x ) = a( x ) + g$.
Тогда, если $y \in I_x$ для некоторого $x$, то $a_g( y ) = a( x ) + g$,
в противном случае $a_g( y ) = a( y )$. При достаточно малых $g$
интервалы $I_x$ определены и не выходят за пределы монотонных участков
функции $a$, прилегающих к $x$ (в частности, не пересекаются),
функция $a_g$ непрерывна, причем монотонность
в каждой точке наследуется от $a$ (иногда становясь нестрогой, что
несущественно). Покажем, что если функция $a$ удовлетворяла неравенству
$a( s ) + a( t ) \ge a( 1 + s + t )$ для всех $s$, $t$, для которых
$-1 \le 1 + s + t \le 1$, то найдется достаточно малое $g$ такое, что и
$a_g$ будет удовлетворять тому же неравенству. Докажем от противного.
Пусть $g_i \to 0$ и для каждого $i$ найдутся $s_i$ и $t_i$, что
$a_{g_i}( s_i ) + a_{g_i}( t_i) < a_{g_i}( 1 + s_i + t_i )$.
Поскольку $( s_i, t_i )$ лежат в компакте, можно, перейдя к
подпоследовательности, получить $s_i \to s$ и $t_i \to t$.
Поскольку $a_{g_i}( x ) \to a( x )$,
$a( s ) + a( t ) = a( 1 + s + t )$.
Докажем от противного, что $1 + s + t \in M$ (и, в частности,
$1 + s + t \in (-1, 1)$). Пусть $1 + s + t \notin M$. Тогда, начиная с
некоторого $i$, будет $a_{g_i}( 1 + s_i + t_i ) = a( 1 + s_i + t_i ) \le
a( s_i ) + a( t_i ) \le a_{g_i}( s_i ) + a_{g_i}( t_i ) <
a_{g_i}( 1 + s_i + t_i )$. Теперь докажем от противного, что
$s, t \in M \cup Z$. Пусть, например, $s \notin M \cup Z$. Тогда
найдется последовательность $\delta _i \to 0$, что
$a( s + \delta _i ) \le a( s )$. Но тогда $a( s + \delta _i ) + a( t )
\ge a( 1 + s + \delta _i + t ) > a( 1 + s + t ) = a( s ) + a( t ) \ge
a( s + \delta _i ) + a( t )$. Здесь $1 + s + \delta _i + t \in (-1, 1)$
начиная с некоторого $i$, так как $1 + s + t \in M$. По той же причине
выполняется второе неравенство.
Таким образом, $s, t \in M \cup Z, 1 + s + t \in M$.
Пусть $s, t \in Z$, тогда $1 + s + t \in Z$, но такого быть не может,
поэтому либо $s$, либо $t$ находится в $M$.
Если $s, t \in M$, получаем $( a( s ) + g_i ) + ( a( t ) + g_i ) =
a_{g_i}( s_i ) + a_{g_i}( t_i ) < a_{g_i}( 1 + s_i + t_i ) =
a( 1 + s + t ) + g_i$, то есть
$a( s ) + a( t ) < a( 1 + s + t )$. Если же, например, $s \in Z, t \in M$, то
$a( s ) + ( a( t ) + g_i ) =
a_{g_i}( s_i ) + a_{g_i}( t_i ) < a_{g_i}( 1 + s_i + t_i ) =
a( 1 + s + t ) + g_i$, то есть опять же
$a( s ) + a( t ) < a( 1 + s + t )$
Тем самым, мы пришли к противоречию,
доказав для достаточно малых $g$ $a_g( s ) + a_g( t ) \ge a_g( 1 + s + t )$.

Обозначим $m: = \min a( M )$,
$b_g( x ) := \frac{m}{m + g} a_g( x ) \nearrow a( x )$.
Тогда можно аналогично
случаю с симметричной перестановкой применить теорему о монотонной сходимости
и получить

\begin{prop}
Если $a(x)$ кусочно-монотонна и удовлетворяет условию (\ref{almostConcave}),
то для любой $u \in {\cal W}(a)$ верно $I(u^*) \le I(u)$.
\end{prop}
