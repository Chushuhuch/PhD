\section{Введение}
Пусть $\Omega = [-1,1]$, 
$F( \cdot, \cdot, \cdot ): \Omega \times \Real \times \Real_+ \to \Real_+$ ---
четная, выпуклая и возрастающая по третьему аргументу непрерывная неотрицательная функция,
$a( \cdot, \cdot ): \Omega \times \Real \to \Real_+$ ---
непрерывная неотрицательная функция. 
Рассмотрим измеримую функцию $u$, заданную на $\Omega$. Для нее верна формула о послойном
представлении: $u( x ) = \int_0^\infty \chi_{\{ x \in [-1,1] :\ u( x ) > t \}} dt - 
\int_{-\infty}^0 \chi_{\{ x \in [-1,1] :\ u( x ) < t \}} dt$.
Определим монотонную перестановку множества $E \subset [-1,1]$: $E^* = [1 - \abs{ E }, 1]$.
Тогда монотонная перестановка функции $u$ определяется следующим образом:
$u^*( x ) = \int_0^\infty \chi_{\{ x \in [-1,1] :\ u( x ) > t \}^*} dt - 
\int_{-\infty}^0 1 - \chi_{\{ x \in [-1,1] :\ u( x ) > t \}^*} dt$.
Аналогично можно определить симметричную перестановку множества $E$ и функции $u$.
$\overline{ E } = [-\frac{\abs{E}}{2}, \frac{\abs{E}}{2}]$,
$\overline{ u }( x ) = \int_0^\infty \chi_{ \overline{ \{ x \in [-1, 1] :\ u( x ) > t \} } } dt -
\int_{-\infty}^0 1 - \chi_{ \overline{ \{ x \in [-1, 1] :\ u( x ) > t \} } } dt$.

Положим $u \in \W(\Omega)$. Введем следующий функционал: $$ I( u ) = \int_\Omega F( u(x),
\abs{ a( x, u( x ) ) u'( x ) } ) dx.$$

Целью работы является доказательство неравенства
\begin{equation}
\label{toprove}
I(u^*) \le I(u).
\end{equation}

Доказательство будет проводиться следующим образом. Сначала неравенство будет
доказано для кусочно-линейных функций $u$. Далее неравенство будет
поэтапно обобщаться путем аппроксимации функций $u$ из широкого класса
последовательностью функций из узкого класса. Сама возможность
аппроксимировать так, чтобы еще и значение функционала сходилось к нужному
значению, есть не всегда. В частности, инфимумы некоторого функционала на
широком классе функций и на узком могут не совпадать. Это явление называется
эффектом Лаврентьева. Тем самым, проводя наши рассуждения, мы показываем, что
эффекта Лаврентьева нет для функционалов данного вида.

В работе \cite{ASC} доказано отсутствие эффекта Лаврентьева для функционалов
вида $\int_{-1}^1 F(u(x), u'(x)) dx$. Мы модифицируем их метод для получения
сходимости в нашем случае.

В работе \cite{Br} показано, что $I(\overline{ u }) \le I( u )$ при условии,
что функция $a$ четна и выпукла по $x$. Однако, в этой работе не учтена и не
исключена возможность возникновения эффекта Лаврентьева. Мы объясняем, почему
его там нет.

Кроме того, мы покажем, что неравенство (\ref{toprove}) выполняется, если вес
$a$ четен и вогнут по $x$ и в некоторых случаях можно ослабить условие
вогнутости.

В работе \cite{Lan} подобное неравенство доказывается для функций $u$,
закрепленных на левом конце: $u( -1 ) = 0$, при условии, что вес убывает.

\section{Свойства весовой функции}

\begin{prop}
Для того, чтобы неравенство $(\ref{toprove})$ выполнянось для любой кусочно-линейной
функции $u$, вес $a$ должен быть четным по $x$: $\forall x, u\ a(x, u)=a(-x, u)$.
\end{prop}

\begin{prop}
Для того, чтобы неравенство $(\ref{toprove})$ выполнянось для любой кусочно-линейной
функции $u$ и любой строго выпуклой $F$, вес $a$ должен удовлетворять условию
$\forall s \le t, u \ a( s, u ) + a( t, u ) \ge a( 1 - t + s, u )$ или, что
эквивалентно при условии четности,
\begin{equation}
\label{almostConcaveSum}
a( s, u ) + a( t, u ) \ge a( 1 + s + t, u )
\end{equation}
\end{prop}

\begin{lm}
Если четная функция $a$ удовлетворяет условию (\ref{almostConcaveSum}), а
также найдется такое $x_0 > -1$, что $a(x_0) = 0$, то $a(-1)=a(1)=0$ и
функция $a$ имеет период, делящий $x_0+1$ нацело. В частности, для любого
неотрицательного
целого $k$ такого, что $-1 + k (x_0 + 1) \le 1$ $a(-1 + k (x_0 + 1)) = 0$.
\end{lm}

\begin{lm}
\label{piecewiseLinearConcave}
Пусть функция $a$ удовлетворяет неравенству (\ref{almostConcaveSum}), $k \in \Nat$.
Тогда кусочно-линейная функция $a_k$, построенная на точках
$(-1 + \frac{2i}{k}, a(-1 + \frac{2i}{k}))$ при $i = 0, 1, \dots, k$, тоже
удовлетворяет (\ref{almostConcaveSum}).
\end{lm}

\begin{lm}
\label{maxConcave}
Пусть функции $a_1$ и $a_2$ удовлетворяют неравенству (\ref{almostConcaveSum}),
тогда функция $a(x) = \max (a_1(x), a_2(x))$ тоже ему удовлетворяет.
\end{lm}

\section{Результат на кусочно-линейных функциях}

Естественным классом для функций $u$ является $\W(-1, 1)$. Однако, нам придется
его немного расширить. Функция $u$ лежит в классе $WD(a)$, где $a$ ---
некоторая функция, удовлетворяющая (\ref{almostConcaveSum}), если найдется
последовательность $-1 = x_0 < x_1 < \dots < x_{n - 1} < x_n = 1$ такая, что
ограничения $u|_{[x_i,x_{i+1}]} \in \W(x_i, x_{i + 1})$ для всех
$0 \le i < n$. В точке $x_i$ ($0 < i < n$) функция $u$ может иметь разрыв, но
только в том случае, если для любого $y \in [u(x_i-), u(x_i+)]$ $a(x_i, y) = 0$.
При интегрировании будем считать вклад точек $x_i$ нулевым.

\begin{thm}
\label{linth}
Пусть функция $a$ неотрицательна, четна и удовлетворяет условию
(\ref{almostConcaveSum}). Тогда, если функция $u \in WD(a)$ и между любыми двумя
разрывами кусочно-линейна, $I(u^*)$ корректно определено $(u^* \in WD(a))$,
$u^*$ кусочно линейна между разрывами и $I(u)\ge I(u^*)$.
\end{thm}

\section{Переход к липшицевым функциям}

\begin{prop}
Пусть вес $a$ непрерывен и для любой кусочно-линейной функции $u$ $I(u^*) \le I(u)$
Тогда для любой липшицевой функции $u$ верно $I(u^*) \le I(u)$.
\end{prop}


\section{Переход к $\W$-функциям}

\begin{prop}
\label{Wapprox}
Если функция $a$ непрерывна, четна и убывает на $[0, 1]$, 
$\forall u \in \W \exists u_n \in Lip: u_n \to u$ в $\W$ и $I(u_n) \to I(u)$.
\end{prop}

\begin{proof}
Заменяем в некоторых местах функцию $u$ на линейную, получая липшицеву.
При увеличении разрешенной производной мера множества замененных точек будет
стремиться к нулю. У каждого липшицевого приближения раздуем вширь участки
линейности так, чтобы наклон на них был ограничен некоторой общей константой.
Мера точек, в которые перешли линейные участки по-прежнему будет стремиться к
$0$. А приближения будут сходиться к $u$. Тогда часть интеграла по замененному
множеству будет сходиться к $0$, а оставшаяся часть --- к $I(u)$ (при этом
существенно используется нужная монотонность веса $a$).
\end{proof}

\rem Те же рассуждения с закреплением функции $u$ на левом конце можно провести на любом интервале по $x$, где
вес $a$ убывает по $x$. То есть получить на этом интервале последовательность $u_h \to u$ в $\W$ и $I( u_h ) \to I( u )$.
Аналогично, можно закрепить правый конец при условии возрастания $a$ по $x$.

\section{Переход к $\W$-функциям в случае симметричной перестановки}

Для каждой $u \in \W$ надем
аппроксимацию липшицевыми $u_h$: $u_h \to u$ в $\W$ и $I( u_h ) \to I( u )$.
Отличие состоит в том, что в случае с монотонной перестановкой можно было закрепить
$u$ в $0$ и аппроксимировать ее отдельно на $[-1,0]$ и $[0,1]$. Поскольку в случае симметричной перестановки вес $a$ имеет
противоположную монотонность, то закреплять нужно на концах интервала. Но при склейке аппроксимаций в $0$ не получится функция
из $\W$, так как склейка может иметь разрыв в $0$.
Для начала, найдем нужную аппроксимацию при условии, что найдется некоторое $t$, что $a( x, u )$ зависит только
от $u$ при $\abs{ x } \le t$.

Найдем точку $x_0 \in [-t, t]$, удовлетворяющую
\begin{equation}
\label{meanConv}
\frac{u(x_0 + \eps) - u(x_0 - \eps)}{2 \eps} \to u'(x_0).
\end{equation}
Зафиксируем $x^-$ и $x^+$ такие, что $x^- + x^+ = 2 x_0$.
Определим $w( x ) = u( x )$ при $x \in [ -1, x^- ] \cup [ x^+, 1 ]$, а на отрезке $[ x^-, x^+ ]$ линейным образом.
Тогда $w \in \W$.
Будем приближать функцию $u$ на интервале $[-1, x_0]$ и отдельно на $[x_0, 1]$.
Получим последовательность липшицевых $u_h^-$, $u_h^+$.
Определим $w_h( x ) = u_h^-( x )$ при $x \le x^-$, $w_h( x ) = u_h^+( x )$ при $x \ge x^+$ и
линейно при $x \in [x^-, x^+]$:
$w_h( x ) = u_h^-( x^- ) + ( u_h^+( x^+ ) - u_h^-( x^- ) ) \frac{ x - x^- }{ x^+ - x^- }$.
Тогда $w_h \to w$ в $\W(-1, 1)$ и $I(w_h) \to I(w)$.

Устремим теперь $x^+$ и $x^-$ к $x_0$, сохраняя соотношение $x^- + x^+ = 2 x_0$.
Тогда (\ref{meanConv}) дает нам ограниченность производной на отрезке
$[ x^-, x^+ ]$ и мы получаем сходимость для произвольной $u$.

Вес $a$ произвольного вида можно приблизить снизу весами с
постоянной площадкой в $0$. Тем самым, по теореме о монотонной сходимости,
получим $I(u) \ge I(\overline{ u })$.

\section{Получение неравенства в случае монотонной перестановки для
$u \in \W$ и кусочно-монотонных весов $a$ вида $a^0( x ) \varkappa( u )$}

Если $a^0$ кусочно-монотонная и в окрестностях минимумов имеет константные
участки, то можно из каждого ее максимума
приближать функцию $u$, в окрестностях минимума заменяя на линейную.
Тем самым, достаточно приблизить описанными весовыми функциями снизу
произвольный кусочно-монотонный.

\begin{prop}
Если $a( x, u ) = a^0( x ) \varkappa( u )$, где $a^0$, $\varkappa$ непрерывны
и неотрицательны, $a^0$ кусочно-монотонна и
удовлетворяет условию (\ref{almostConcaveSum}), то для любой
$u \in \W(\Omega \setminus Z^0)$
$I(u^*) \le I(u)$.
\end{prop}

\section{Избавление от требования кусочной монотонности $a$}

Получим теперь неравенство для произвольных функций $a( x, u )$ вида
$a^0( x ) \varkappa( u )$.
Приблизим $a^0$ кусочно-линейными. Каждую из приближающих функций домножаем
так, чтобы она была ниже $a^0$ вне окрестностей концов отрезка.
Максимум нескольких первых будет монотонно приближать $a^0$ вне окрестностей
концов отрезка. На концах отрезка делаем $u'=0$, тем самым добиваясь
монотонной сходимости интеграла. Далее, переходим к произвольной $u$.

\begin{prop}
Если функция $a(x,u)$ представима в виде $a^0(x) \varkappa(u)$, где
$a^0$ удовлетворяет условию (\ref{almostConcaveSum}), то для любой
$u \in \W(-1,1)$ $I(u^*) \le I(u)$.
\end{prop}
