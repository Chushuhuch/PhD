\section{Избавление от требования кусочной монотонности $a$}

!!! Переписать для произвольных $a( x, u )$ !!!

Получим теперь неравенство для произвольных функций $a( x, u )$ вида
$a^0( x ) \varkappa( u )$, но для пространства
функций $u \in W(-1, 1)$, постоянных в $\eps$-окрестности $\pm 1$.

Зафиксируем некоторое $\eps$ и некоторую функцию $u \in \W(-1, 1)$, постоянную
на $[-1,-1 + \eps] \cup [1 - \eps, 1]$. Предположим, что
$\int_{-1}^1 F(u(x), a(x, u) u'(x))dx < \infty$. Возьмем число $\theta < 1$ и
число $k(\theta) \in \Nat$ так, чтобы $\theta a_k^0(x) \le a^0(x)$ при
$x \in [-1 + \eps, 1 - \eps]$. Здесь $a_k^0$ --- кусочно-линейное приближение
$a^0$, состоящее из $k$ отрезков, которое, в соответствии с леммой
(\ref{piecewiseLinearConcave}), удовлетворяет неравенству
(\ref{almostConcaveSum}). Покажем, что для любого
$\theta$ найдется такое $k$. Для любого $i$ будет выполнено
$\theta a_k^0(-1 + \frac{2i}{k}) < a^0(-1 + \frac{2i}{k})$. Для любого
неотрицательного $z < \frac{2}{k}$ по неравенству (\ref{almostConcaveSum})
$a^0(-1 + z) + a^0(1 - \frac{2i}{k} - z) \ge a^0(1 - \frac{2i}{k})$, то есть
$a^0(-1 + \frac{2i}{k} + z) \ge a^0(-1 + \frac{2i}{k}) - a^0(-1 + z)$, так как
$a^0$ четная. С другой стороны, по неравенству (\ref{almostConcaveSum})
$a_k^0(-1 + \frac{2i}{k} + z) \le a_k^0(-1 + \frac{2i}{k}) + a_k^0(-1 + z)$.
Тем самым, $a^0(-1 + \frac{2i}{k} + z) - \theta a_k^0(-1 + \frac{2i}{k} + z)
\ge (1 - \theta) a^0(-1 + \frac{2i}{k}) - a^0(-1 + z) - \theta a_k^0(-1 + z)$.
Но $a^0(-1 + z) + \theta a_k^0(-1 + z) \le
2 \max \{ a^0(-1 + \zeta): \zeta \in [0, \frac{2}{k}]\} \to 0$.
Оценим снизу $a^0(-1 + \frac{2i}{k})$. Поскольку нас интересует неравенство на
$[-1 + \eps, 1 - \eps]$, нам достаточно оценки для
$-1 + \frac{2i}{k} \in [-1 + \eps - \frac{2}{k}, 1 - \eps + \frac{2}{k}]$.
А начиная с некоторого $k$, достаточно получить оценку при
$-1 + \frac{2i}{k} \in [-1 + \frac{\eps}{2}, 1 - \frac{\eps}{2}]$.
$a^0(-1 + \frac{2i}{k}) \ge
\min \{ a^0(x): x \in [-1 + \frac{\eps}{2}, 1 - \frac{\eps}{2}] \}$,
что положительно и не зависит от $k$.
Значит, найдется $k(\eps, \theta)$ такое, что $\theta a_k^0(x) \le a^0(x)$ при
$x \in [-1 + \eps, 1 - \eps]$. Положим $\theta_j = 1 - \frac{1}{j}$,
$b_j(x) = \max\limits_{i \le j} \theta_i a_{k(\theta_i)}^0(x)$ и
$a_j(x, u) = \varkappa(u) b_j(x)$. По лемме (\ref{maxConcave}) $b_j$
удовлетворяют неравенству (\ref{almostConcaveSum}).
$a_j(x, u) \le a(x, u)$ при $x \in [-1 + \eps, 1 - \eps]$ и
$a_j(x, u) \nearrow a(x, u)$.

Функции $b_j$ кусочно-монотонные как максимум нескольких кусочно-монотонных
функций, поэтому
\begin{equation}
\label{inequlityForPartiallyLinear}
\int_{-1}^1 F(u(x), a_j(x, u) u'(x)) dx \ge
\int_{-1+\eps}^{1-\eps} F(u^*(x), a_j(x, u) {u^*}'(x)) dx.
\end{equation}
Покажем, что 
\begin{align*}
\int_{-1}^1 F(u(x), a_j(x, u) u'(x)) dx \to &
\int_{-1}^1 F(u(x), a(x, u) u'(x)) dx \\
\int_{-1+\eps}^{1-\eps} F(u^*(x), a_j(x, u) {u^*}'(x)) dx \to &
\int_{-1+\eps}^{1-\eps} F(u^*(x), a(x, u) {u^*}'(x)) dx.
\end{align*}
В обоих случаях подынтегральная функция левой части интегрируема,
монотонно возрастает по $j$ и сходится к подынтегральной функции правой части.
Тем самым, по теореме о монотонной сходимости получаем
$$I(u) = \int_{-1}^1 F(u(x), a(x, u) u'(x)) dx \ge
\int_{-1+\eps}^{1-\eps} F(u^*(x), a(x, u) {u^*}'(x)) dx.$$

Возьмем теперь произвольную $u \in \W(-1,1)$ и $u_\eps(x) = u(x)$ при
$x \in [-1 + \eps, 1 - \eps]$ и постоянную при остальных $x$. Тогда
$u_\eps \to u$ в $\W(-1,1)$ и $I(u_\eps) \to I(u)$. Поясним последнее.
$u_\eps \to u$ поточечно. Кроме того,
$I(u_\eps)=\eps F(u(-1 + \eps), 0) +
\int_{-1+\eps}^{1 - \eps} F(u(x), a(x) u'(x))dx + \eps F(u(1 - \eps), 0) \le
I(u) + 2\eps \max\{F(u(x), 0): x \in [-1,1]\} < \infty$. То есть, по теореме о
мажорируемой сходимости $I(u_\eps) \to I(u)$.
Поскольку $u_\eps \to u$ в $\W(-1+\eps,1-\eps)$,
$u_\eps \rightharpoondown u$ в $\W(-1+\eps,1-\eps)$. Значит
$\int_{-1+\eps}^{1-\eps} F(u^*(x), a(x, u) {u^*}'(x)) dx \le
\varliminf
\int_{-1+\eps}^{1-\eps} F(u_\eps^*(x), a(x, u_\eps) {u_\eps^*}'(x)) dx$.
Тем самым,
$$I(u) \ge \int_{-1+\eps}^{1-\eps} F(u^*(x), a(x, u) {u^*}'(x)) dx$$
уже для произвольно функции $u \in \W(-1,1)$. Но $\eps$ здесь можно взять
любое, поэтому получаем

\begin{prop}
Если функция $a(x,u)$ представима в виде $a^0(x) \varkappa(u)$, где
$a^0$ удовлетворяет условию (\ref{almostConcaveSum}), то для любой
$u \in \W(-1,1)$ $I(u^*) \le I(u)$.
\end{prop}
