\section{Результат на кусочно-линейных функциях}

Рассмотрим случай, когда $a$ не зависит от $u$. Докажем наше неравенство для кусочно-линейных функций,
разрешив им иметь разрывы в множестве $Z = a^{-1}(0)$ нулей функции $a$. При интегрировании будем
считать вклад этих точек нулевым.

\begin{thm}
\label{linth}
Рассмотрим одномерный случай: $n = 1$, $\Omega = [-1, 1]$.
Пусть функция $a$ зависит только от $x$, неотрицательна, четна и удовлетворяет условию (\ref{almostConcave}).
Тогда, если $u$ --- кусочно-линейная функция с разрывами лишь в точках множества $Z$, то функция $u^*$
тоже будет иметь разрывы только на множестве $Z$ и $I(u)\ge I(u^*)$.
\end{thm}

\begin{proof}
Положим $-1 = x_1 < x_2 < \dots < x_K = 1$ --- множество точек перелома функции $u$, включая точки
разрывов. Рассмотрим множество $U$ образов функции $u$, не являющихся образами конечных точек
линейных участков: $U := u( \Omega ) \setminus \cup_{i = 2}^K \{ \lim\limits_{x \nearrow x_i} u(x) \}
\setminus \cup_{i = 1}^{K - 1} \{ \lim\limits_{x \searrow x_i} u(x) \}$. Множество $U$ можно
разбить на конечное число связных областей $U = \cup_{j = 1}^N G_j$. При $u_0 \in G_j$ число решений уравнения
$u(y) = u_0$ не зависит от $u_0$ и в этой области задается набором линейных функций: $y_k^j(u_0) = y$,
$k = 1 \dots m_j$ (расположим их в порядке увеличения значений). При этом будем
считать, что каждому разрыву соответствует вертикальный участок графика, а
также некоторое решение $y_k^j(u)$, постоянное по $u$. При $y \in u^{-1}(U)$
$y_k^j{}'(u(y))=\frac{1}{u'(y)}$, если в точке $y$ разрыв, то считаем
$y_k^j{}'(u(y))=0$.

Решение уравнения $u^*(y^*)=u_0$ ($u_0 \in U$) можно выразить используя решения $y_k^j$.
Если $u(-1)<u_0$ и $m_j$ четное, то $y^*=1-\sum_{k=1}^{m_j} (-1)^k y_k^j$, если нечетное, то
$y^*=-\sum_{k=1}^{m_j} (-1)^k y_k^j$, а в случае $u(-1)>u_0$ $y^*=-1+\sum_{k=1}^{m_j} (-1)^k y_k^j$
при четном $m_j$ и $y^*=\sum_{k=1}^{m_j} (-1)^k y_k^j$ при нечетном.
Будем за $y^*$ обозначать $(u^*)^{-1}$ (при этом разрывы переходят в константные участки и наоборот).
Тогда ${y^*}'(u) = \sum_{k=1}^{m_j} \abs{y_k^j{}'(u)}$ при $u \in G_j$, поскольку ${y^*}'(u)\ge 0$.
Отсюда ясно, что константный кусок у
$y^*( u )$ может быть только при тех $u$, при которых все $y_k^j(u)$ константные. То есть все
$y_k^j(u)$ --- точки разрыва, а значит расстояние от $-1$ до каждой из них кратно периоду. Тогда ясно, что
расстояние от $-1$ до $y^*(u)$ тоже кратно периоду, то есть $y^*(u) \in Z$. Тем самым, $u^*$ может иметь
разрывы лишь в точках множества $Z$. Перейдем к доказательству неравенства.

Множества нулей $u'$ и ${u^*}'$ могут иметь ненулевую меру. Однако, их меры
совпадают. Более того, меры множеств их нулей при фиксированном значении
функции тоже совпадают. Тогда
$$\int_{(u')^{-1} (0)} F( u(x), 0 ) dx = \int_{({u^*}')^{-1} (0)} F( u^*(x), 0 ) dx =: C.$$
Рассмотрим оставшиеся части интегралов:
\begin{multline*}
I(u) - C = \sum_{j=1}^N \int_{u^{-1}(G_j)} F( u(x), \abs{ a(x) u'(x) } ) dx =
\\ \sum_{j=1}^N \int_{G_j} \sum_{k=1}^{m_j} F( u, \frac{ a(y_k^j(u)) }{ \bigabs{y_k^j{}'(u)} } ) \bigabs{y_k^j{}'(u)} du,
\end{multline*}
\begin{multline*}
I(u^*) - C = \sum_{j=1}^N \int_{(u^*)^{-1}(G_j)} F( u^*(x), \bigabs{ a(x) {u^*}'(x) } ) dx = 
\\ \sum_{j=1}^N \int_{G_j} F( u^*, \frac{ a(y^*(u^*)) }{\sum_{k=1}^{m_j} \bigabs{ y_k^j{}'(u^*) }} )
\sum_{k=1}^{m_j} \bigabs{ y_k^j{}'(u^*) } du^*.
\end{multline*}
Зафиксируем $j$ и $u = u^*$ в правых частях и докажем
неравенство для подынтегральных выражений. Обозначим
$b_k=|y_k^j{}'(u)|$, $y_k=y_k^j(u)$, $y^*=y^*(u)$, $m = m_j$. Тогда
требуемое утверждение имеет вид:
$$T:=\sum_{k=1}^m b_k F( \frac{ a(y_k) }{b_k} ) \ge F( \frac{ a(y^*) }{ \sum_{k=1}^m b_k  } ) \sum_{k=1}^m b_k$$
С помощью неравенства Йенсена для функции $F$ получаем
$$T \ge F( \frac{ \sum_{k=1}^m a(y_k) }{ \sum_{k=1}^m b_k } ) \sum_{k=1}^m b_k.$$
Тогда нам достаточно установить $\sum_{k=1}^m a(y_k) \ge a(y^*)$, что показано в лемме.
\end{proof}

\rem Если снять условие четности с функции $a$, то лемма тем не менее будет выполняться. Если теперь положить
$u( -1 ) = 0$, то леммы будет достаточно для выполнения теоремы, поскольку в доказательстве всегда будет
$u( -1 ) < u_0$ и нам понадобятся только два неравенства из четырех, как раз те, которые обеспечивает лемма.
В работе \cite{Lan} результат доказывается для убывающих функций $a$. Несложно заметить, что
условие леммы выполняется для убывающих функций, поскольку $s \le 1 - t + s$, а значит, и
$a( s ) + a( t ) \ge a( 1 - t + s )$.
