\section{Результат на кусочно-линейных функциях}

Докажем наше неравенство для кусочно-линейных функций.

\begin{thm}
\label{linth}
Пусть функция $a$ неотрицательна, четна и удовлетворяет условию (\ref{almostConcave}).
Тогда, если $u$ --- кусочно-линейная функция, то $I(u) \ge I(u^*)$.
\end{thm}

\begin{proof}
Положим $-1 = x_1 < x_2 < \dots < x_K = 1$ --- множество точек перелома функции $u$.
Рассмотрим множество $U$ образов функции $u$, не являющихся образами конечных точек
линейных участков: $U := u( [-1, 1] ) \setminus \{ u(x_1), \dots, u(x_K) \}$.
Множество $U$ можно разбить на конечное число связных областей $U = \cup_{j = 1}^N G_j$.

При $u_0 \in G_j$ число решений уравнения $u(y) = u_0$ не зависит от $u_0$
и в этой области задается набором линейных функций: $y_k^j(u_0) = y$,
$k = 1 \dots m_j$ (расположим их в порядке увеличения значений).
$y_k^j{}'(u(y))=\frac{1}{u'(y)}$ при $y \in u^{-1}(U)$.

Решение уравнения $u^*(y^*)=u_0$ ($u_0 \in U$) можно выразить используя решения $y_k^j$.
Если $u(-1)<u_0$ и $m_j$ четное, то $y^*=1-\sum_{k=1}^{m_j} (-1)^k y_k^j$, если нечетное, то
$y^*=-\sum_{k=1}^{m_j} (-1)^k y_k^j$, а в случае $u(-1)>u_0$ $y^*=-1+\sum_{k=1}^{m_j} (-1)^k y_k^j$
при четном $m_j$ и $y^*=\sum_{k=1}^{m_j} (-1)^k y_k^j$ при нечетном.
Будем за $y^*(v)$ обозначать $(u^*)^{-1}(v)$.
Тогда ${y^*}'(v) = \sum_{k=1}^{m_j} \abs{y_k^j{}'(v)}$ при $u \in G_j$, поскольку знаки в выражении для
$y^*$ и знаки $y_k^j$ чередуются и ${y^*}'(v)\ge 0$.

Множества нулей $u'(x)$ и ${u^*}'(x)$ могут иметь ненулевую меру.
Однако, они не вносят вклада в интеграл, поскольку $F(u(x), 0) = 0$.

Рассмотрим оставшиеся части интегралов:
\begin{multline*}
I(u) = \sum_{j=1}^N \int_{u^{-1}(G_j)} F(u(x), \abs{a(x, u(x)) u'(x)}) dx =
\\ \sum_{j=1}^N \int_{G_j} \sum_{k=1}^{m_j} F(v, \frac{a(y_k^j(v), v)}{\bigabs{y_k^j{}'(v)}}) \bigabs{y_k^j{}'(v)} dv,
\end{multline*}
\begin{multline*}
I(u^*) = \sum_{j=1}^N \int_{(u^*)^{-1}(G_j)} F(u^*(x), \bigabs{a(x, u(x)) {u^*}'(x)}) dx = 
\\ \sum_{j=1}^N \int_{G_j} F(v, \frac{a(y^*(v), v)}{\sum_{k=1}^{m_j} \bigabs{y_k^j{}'(v)}})
\sum_{k=1}^{m_j} \bigabs{ y_k^j{}'(v) } dv.
\end{multline*}

Зафиксируем $j$ и $v$ в правых частях и докажем неравенство для подынтегральных выражений.
Обозначим $b_k := |y_k^j{}'(v)|$, $y_k := y_k^j(v)$, $y^* := y^*(v)$, $m := m_j$. Тогда
требуемое утверждение имеет вид:
$$T:=\sum_{k=1}^m b_k F( \frac{ a(y_k, v) }{b_k} ) \ge F( \frac{ a(y^*, v) }{ \sum_{k=1}^m b_k  } ) \sum_{k=1}^m b_k$$
С помощью неравенства Йенсена для функции $F$ получаем
$$T \ge F( \frac{ \sum_{k=1}^m a(y_k, v) }{ \sum_{k=1}^m b_k } ) \sum_{k=1}^m b_k.$$
Тогда нам достаточно установить $\sum_{k=1}^m a(y_k, v) \ge a(y^*, v)$, что верно с учетом леммы \ref{weightSum} и замечания
после нее.
\end{proof}

\rem Если положить $u( -1 ) = \min\limits_{[-1, 1]} u$, то леммы \ref{weightSum} будет достаточно для выполнения теоремы, поскольку
$u( -1 ) < u_0$ и нам понадобятся только два неравенства из четырех, как раз те, которые обеспечивает лемма.
В работе \cite{Lan} результат доказывается для убывающих функций $a$. Несложно заметить, что
условие леммы выполняется для убывающих функций, поскольку $s \le 1 - t + s$, а значит и
$a( s ) + a( t ) \ge a( 1 - t + s )$.
