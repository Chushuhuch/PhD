\section{Введение}
Пусть $F( \cdot, \cdot ): [-1, 1] \times \Real_+ \to \Real_+$ ---
непрерывная неотрицательная функция, выпуклая и возрастающая по второму аргументу.
$a( \cdot, \cdot ): [-1, 1] \times \Real \to \Real_+$ --- непрерывная функция.
Рассмотрим измеримую функцию $u$, заданную на $[-1, 1]$.
Для нее верна формула о послойном представлении:
$$u( x ) = \int_0^\infty \chi_{\{ x \in [-1,1] :\ u( x ) > t \}} dt - 
\int_{-\infty}^0 \chi_{\{ x \in [-1,1] :\ u( x ) < t \}} dt.$$
Определим монотонную перестановку множества $E \subset [-1,1]$: $E^* = [1 - \abs{ E }, 1]$.
Тогда монотонная перестановка функции $u$ определяется следующим образом:
$u^*( x ) = \int_0^\infty \chi_{\{ x \in [-1,1] :\ u( x ) > t \}^*} dt - 
\int_{-\infty}^0 1 - \chi_{\{ x \in [-1,1] :\ u( x ) > t \}^*} dt$.
Аналогично можно определить симметричную перестановку множества $E$ и функции $u$.
$\overline{ E } = [-\frac{\abs{E}}{2}, \frac{\abs{E}}{2}]$,
$\overline{ u }( x ) = \int_0^\infty \chi_{ \overline{ \{ x \in [-1, 1] :\ u( x ) > t \} } } dt -
\int_{-\infty}^0 1 - \chi_{ \overline{ \{ x \in [-1, 1] :\ u( x ) > t \} } } dt$.

Положим $u \in \W(-1, 1)$.
Введем следующий функционал: $$ I( u ) = \int_\Omega F( u(x), \abs{ a( x, u( x ) ) u'( x ) } ) dx.$$

Целью работы является доказательство неравенства
\begin{equation}
\label{toprove}
I(u^*) \le I(u).
\end{equation}

В работе \cite{Br} показано, что $I(\overline{ u }) \le I( u )$ при условии, что функция $a$ четна и выпукла по $x$,
а функция $u$ липшицева.
Мы завершаем доказательство Брока для произвольных $u \in \W(-1, 1)$, а также устанавливаем необходимость
четности и выпуклости для выполнения неравенства.

Мы находим необходимое условие на вес $a$ для выполнения неравенства (\ref{toprove}) и доказываем его достаточность.
В частности, неравенство выполнено, если вес $a$ четен и вогнут по $x$.

Не умаляя общности, можно считать, что $u( x ) \ge 0$ всюду.
В работе \cite{Lan} подобное неравенство доказывается для функций $u$,
закрепленных на левом конце: $u( -1 ) = 0$, при условии, что вес убывает.
