\section{Получение неравенства в случае монотонной перестановки для
$u \in \W$ и кусочно-монотонных по $x$ весов $a$}

Зафиксируем некоторый вес $a$ и некоторую функцию $u \in {\cal W}(a)$. 
Предположим, что вес $a$ удовлетворяет следующим условиям:

\begin{subequations}
\label{goodWeight}
\begin{multline}
\label{pieceMono}
\exists \eps > 0,\ -1 = x_0 < x_1 < \dots < x_k = 1:\\ a( \cdot, v )
\text{ монотонна при } x \in [x_i, x_{i + 1}], \abs{ v - u(x) } \le \eps
\end{multline}
\begin{multline}
\label{flatMin}
a( x_i, u(x_i) ) \neq 0,\ x_i \text{ --- точка локального минимума} \Rightarrow \\
\exists \eps > 0:\ a( x, v ) \equiv a( v ) \text{ при }
\abs{x - x_i} \le \eps, \abs{v - u(x_i)} \le \eps
\end{multline}
\begin{equation}
\label{wideZero}
a( x, u(x) ) = 0 \Rightarrow \exists \eps > 0 : a( x, [u(x) - \eps, u(x) + \eps] ) = 0
\end{equation}
\end{subequations}

Тогда, используя рассуждения из утверждения \ref{Wapprox} и параграфа \ref{sobolevSymm},
можно построить последовательность
$u_n \in {\cal W}(a)$ такую, что $u_n \to u$ в ${\cal W}(a)$ и $I(u_n) \to I(u)$.
И, тем самым, доказать неравенство (\ref{toprove}) для функции $u$.

Действительно, следуя построению в параграфе \ref{sobolevSymm},
можно приближать функцию $u$ из локальных максимумов функции $a( x, u(x) )$.
При этом, в нулевых локальных
минимумах $a( x, u(x) )$ функции $u_n$ будут иметь разрывы, которые, однако, не
выводят их из класса ${\cal W}(a)$ (это следует из того факта, что
на каждом из отрезков $[x_i, x_{i + 1}]$ функции $u_n$ сходятся к $u$ равномерно
и условия (\ref{wideZero})). А в ненулевых локальных минимумах выполняется
условие (\ref{flatMin}), поэтому можно, предварительно передвинув
соответствующий $x_i$ в подходящую точку, устранить разрыв, заменив функцию
на линейную в малой окрестности $x_i$.

Остается свести задачу с произвольным весом $a$ к задаче с весом,
удовлетворяющим условиям (\ref{goodWeight}). Для начала рассмотрим случай
весовой функции $a(x)$, не зависящей от второго аргумента.
