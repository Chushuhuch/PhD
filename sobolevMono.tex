\section{Получение неравенства в случае монотонной перестановки для
$u \in \W$ и кусочно-монотонных весов $a$ вида $a^0( x ) \varkappa( u )$}

Предположим, что верно представление $a( x, u ) = a^0( x ) \varkappa( u )$,
где $a^0$ и $\varkappa$ непрерывны и неотрицательны.
Далее, предположим, что отрезок $[-1, 1]$
можно разделить на конечное число промежутков, на каждом из которых $a^0$
монотонна. На первом (возможно пустом) из этих промежутков $a^0$ будет
возрастать, на втором
убывать, далее чередоваться и, наконец, на последнем убывать.
Предположим, что в окрестностях всех точек минимумов функции $a^0$ кроме
$-1$ и $1$ функция $a$ не зависит от $x$. Тогда, если взять соседние
возрастающий и убывающий куски, с ними можно будет повторить процедуру
аппроксимации произвольной $u \in \W$ как это было сделано для
симметричных перестановок. А именно: приближать функцию
$u$ на каждом из участков монотонности, закрепив в точке максимума $a^0$,
а затем склеивать эти приближения в некоторой подходящей точке функции $u'$,
лежащей на участке плоского минимума функции $a^0$.
Поскольку для крайних точек отрезка, а также множества $Z^0$ нулей
функции $a^0$ вопрос склейки не
стоит, не нужно требовать от $a^0$ постоянства в их окрестностях, достаточно
просто аппроксимировать $u$, закрепив в ближайшем максимуме $a^0$.

Остался вопрос о том, как приблизить произвольную функцию
$a( x, u ) = a^0( x ) \varkappa( u )$ снизу такими же, но с плоскими по $x$
минимумами. Введем обозначение для множества локальных строгих ненулевых
минимумов функции $a^0$
$M = \{ x \in ( -1, 1 ) \setminus Z^0: \exists \eps( x ) \forall y :
0 < \abs{ x - y } < \eps \ a^0( y ) > a^0( x ) \}$.
Пусть $a^0_g$ ($g \ge 0$) это функция, полученная из $a^0$ заменой на
константу в окрестностях этих точек так, что сам минимум поднимается на $g$.
А именно: пусть $x \in M$ и $I_x = ( l_x, r_x ) \ni x$ такой интервал, что
$a^0\mathop{|}_{I_x} < a^0( x ) + g$ и
$a^0( l_x ) = a^0( r_x ) = a^0( x ) + g$.
Тогда, если $y \in I_x$ для некоторого $x$, то $a^0_g( y ) = a^0( x ) + g$,
в противном случае $a^0_g( y ) = a^0( y )$. При достаточно малых $g$
интервалы $I_x$ определены и не выходят за пределы монотонных участков
функции $a^0$, прилегающих к $x$ (в частности, не пересекаются),
функция $a^0_g$ непрерывна, причем монотонность
в каждой точке наследуется от $a^0$ (иногда становясь нестрогой, что
несущественно). Покажем, что если функция $a^0$ удовлетворяла неравенству
$a^0( s ) + a^0( t ) \ge a^0( 1 + s + t )$ для всех $s$, $t$, для которых
$-1 \le 1 + s + t \le 1$, то найдется достаточно малое $g$ такое, что и
$a^0_g$ будет удовлетворять тому же неравенству. Докажем от противного.
Пусть $g_i \to 0$ и для каждого $i$ найдутся $s_i$ и $t_i$, что
$a^0_{g_i}( s_i ) + a^0_{g_i}( t_i) < a^0_{g_i}( 1 + s_i + t_i )$.
Поскольку $( s_i, t_i )$ лежат в компакте, можно, перейдя к
подпоследовательности, получить $s_i \to s$ и $t_i \to t$.
Поскольку $a^0_{g_i}( x ) \to a^0( x )$,
$a^0( s ) + a^0( t ) = a^0( 1 + s + t )$.
Докажем от противного, что $1 + s + t \in M$ (и, в частности,
$1 + s + t \in (-1, 1)$). Пусть $1 + s + t \notin M$. Тогда, начиная с
некоторого $i$, будет $a^0_{g_i}( 1 + s_i + t_i ) = a^0( 1 + s_i + t_i ) \le
a^0( s_i ) + a^0( t_i ) \le a^0_{g_i}( s_i ) + a^0_{g_i}( t_i ) <
a^0_{g_i}( 1 + s_i + t_i )$. Теперь докажем от противного, что
$s, t \in M \cup Z^0$. Пусть, например, $s \notin M \cup Z^0$. Тогда
найдется последовательность $\delta _i \to 0$, что
$a^0( s + \delta _i ) \le a^0( s )$. Но тогда $a^0( s + \delta _i ) + a^0( t )
\ge a^0( 1 + s + \delta _i + t ) > a^0( 1 + s + t ) = a^0( s ) + a^0( t ) \ge
a^0( s + \delta _i ) + a^0( t )$. Здесь $1 + s + \delta _i + t \in (-1, 1)$
начиная с некоторого $i$, так как $1 + s + t \in M$. По той же причине
выполняется второе неравенство.
Таким образом, $s, t \in M \cup Z^0, 1 + s + t \in M$.
Пусть $s, t \in Z^0$, тогда $1 + s + t \in Z^0$, но такого быть не может,
поэтому либо $s$, либо $t$ находится в $M$.
Если $s, t \in M$, получаем $( a^0( s ) + g_i ) + ( a^0( t ) + g_i ) =
a^0_{g_i}( s_i ) + a^0_{g_i}( t_i ) < a^0_{g_i}( 1 + s_i + t_i ) =
a^0( 1 + s + t ) + g_i$, то есть
$a^0( s ) + a^0( t ) < a^0( 1 + s + t )$. Если же, например, $s \in Z^0, t \in M$, то
$a^0( s ) + ( a^0( t ) + g_i ) =
a^0_{g_i}( s_i ) + a^0_{g_i}( t_i ) < a^0_{g_i}( 1 + s_i + t_i ) =
a^0( 1 + s + t ) + g_i$, то есть опять же
$a^0( s ) + a^0( t ) < a^0( 1 + s + t )$
Тем самым, мы пришли к противоречию,
доказав для достаточно малых $g$ $a^0_g( s ) + a^0_g( t ) \ge a^0_g( 1 + s + t )$.

Обозначим $m: = \min a^0( M )$,
$b^0_g( x ) := \frac{m}{m + g} a^0_g( x ) \nearrow a^0( x )$.
Соответственно, $b_g( x, u ) = b^0_g( x ) \varkappa( u ) \nearrow a( x, u )$.
Тогда можно аналогично
случаю с симметричной перестановкой применить теорему о монотонной сходимости
и получить

\begin{prop}
Если $a( x, u ) = a^0( x ) \varkappa( u )$, где $a^0$, $\varkappa$ непрерывны
и неотрицательны, $a^0$ кусочно-монотонна и
удовлетворяет условию (\ref{almostConcave}), то для любой
$u \in \W(\Omega \setminus Z^0)$
$I(u^*) \le I(u)$.
\end{prop}
